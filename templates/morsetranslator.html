Morse Code Translator



The Morse code translator can translate both to and from Morse code, and is compatible with most online Morse translators.



The Morse Code translator was written independently, primarily to showcase in this portfolio. Originally, the scope of the project was to translate from plaintext to Morse Code. As an additional challenge, I decided to build functionality that allows translating from Morse to plaintext as well. The end result is a translator that can go both ways - both to and from Morse.

Translating to Morse is as simple as finding the given letter (key) in the alphabet dictionary. The value is then returned as equivalent to the key, and when the entire string is processed, it is returned as Morse code.

Translating from Morse is a lot more complex, which was a satisfying challenge. The program will collect symbols (dots and dashes) until it reaches a break. This break can be indicated by a slash or spaces, the most common syntax for indicating breaks in written Morse.

When the program has detected a break, it will group all previously collected symbols together and find their matching key in the dictionary (which for this purpose, is reversed). Upon locating this key, it amends the value to the final translation. When this entire process is complete, the result is a fully translated string.
