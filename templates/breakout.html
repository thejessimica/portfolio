Breakout



A re-creation of the 80's game Breakout, written in Python using primarily the Turtle library.



For this project, a strong basis of OOP was used to form the different game objects and their various methods.

When the game starts, it generates a set amount of blocks and fills a list with their coordinates. This list is stored for later use.

The player's paddle at the bottom of the screen is controlled with either A and D or the arrow keys. Both the paddle and ball are separate classes.

While the game is running, it will loop through a check to see if the ball is close to any coordinates in the list of blocks. If it is, the block is moved to -1000, 1000, and a sound plays. The Pygame module is used to play the sound effects without stopping the code or using threads.
